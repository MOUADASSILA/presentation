Qu'est-ce que le "Développement Web" ?

  • Exemple: La construction d'une maison. Il y a l'Architecte (le Designer) et le Constructeur (le Développeur).
  • L'Architecte (Le Designer): C'est le UI/UX Designer. Il dessine les plans et pense à l'UX/UI.
  • Le Constructeur (Le Développeur): Il prend les plans et construit la maison avec des outils (langages de programmation).

Pourquoi le Développement Web est-il Important ?

  • Communication : Le web permet aux entreprises et aux gens de communiquer (Blogs, Médias Sociaux).
  • E-commerce : La vente et l'achat en ligne. Le web est le plus grand marché du monde.
  • Éducation : Apprendre n'importe quoi en ligne (YouTube, Coursera, Cours).
  • Gestion (Business) : Les entreprises utilisent des outils sur le web pour organiser leur travail.

Comment fonctionne un "Site Web" ?

  • C'est un ensemble de fichiers (code, images) stockés sur un **"Serveur"**.
  • Serveur : Un ordinateur puissant qui fonctionne 24/7 et qui est connecté à Internet.
  • Client (Votre navigateur) : Envoie une requête (demande) au Serveur.
  • Serveur : Répond et envoie les fichiers au client.
  • Client : Assemble les fichiers et vous affiche la page.

Les 5 Étapes de la Création d'un Site

Étape 1: Planification (Planning)

  • POURQUOI ? (L'Objectif): Qu'est-ce que ce site va faire ? (Vente ? Éducation ?...).
  • POUR QUI ? (L'Audience): Qui sont les gens qui vont l'utiliser ?
  • QUOI ? (Les Fonctionnalités): Quelles sont les choses qu'il doit faire ? (Connexion ? Formulaire ? Galerie ?).

Étape 2: Design (UI & UX)

UI (User Interface / Interface Utilisateur)

Comment il apparaît (L'aspect visuel). Les couleurs, la typographie, les images. Objectif : Qu'il soit Beau.

UX (User Experience / Expérience Utilisateur)

Comment il fonctionne (L'expérience). Est-il facile à utiliser ? Objectif : Qu'il soit Facile.

Étape 3 : Développement (Le Code)

Frontend (L'Interface / La Façade)

  • Tout ce que l'utilisateur voit et avec quoi il interagit dans le navigateur.
  • Exemple: Le salon de la maison. Il contient le décor, les canapés, la télévision.
  • Langages Fondamentaux: HTML, CSS, JavaScript.

Backend (Les Coulisses / L'Arrière-plan)

  • Le moteur du site. C'est ce qui ne se voit pas, et qui fonctionne sur le Serveur.
  • Exemple: La cuisine d'un restaurant. C'est elle qui prépare les plats et stocke les ingrédients.
  • Fonction: Il stocke les informations (utilisateurs) dans une "Database" (base de données), et il fait les calculs compliqués.

Étape 4: Test

  • On teste s'il contient des problèmes (Bugs).
  • Est-ce que tout fonctionne bien ? Les boutons ? Les formulaires ?
  • Est-ce qu'il s'affiche bien sur un téléphone ? (C'est le Responsive Design).

Étape 5: Publication (Hébergement, Domaine, SSL)

  • Hosting (Hébergement): Louer un espace sur un Serveur pour y mettre vos fichiers 24h/24 et 7j/7.
  • Domain Name (Nom de Domaine): C'est l'adresse facile (par ex : google.com). Il vous redirige vers l'IP du Serveur.
  • Certificat SSL: Il sécurise la connexion. Il change http:// en https://. Très important pour la confiance et la sécurité.

Les Outils Fondamentaux

HTML

Le Squelette (Structure).

CSS

Les Vêtements et le Décor (Style).

JavaScript

Le Cerveau et les Muscles (Interaction).

1. HTML (Le Squelette)

Qu'est-ce que c'est ?

C'est le squelette. Il ne contrôle pas les couleurs, seulement la structure. Il dit au navigateur :

  • "Ici, c'est un titre"
  • "Ici, c'est une image"
  • "Ici, c'est un bouton"

Exemples de "Tags"

<h1>Grand Titre</h1>
<p>Ceci est un paragraphe.</p>
<img src="image.jpg">
<a href="page2.html">Lien</a>
<div>Boite (Container)</div>
<button>Cliquez Ici</button>

2. CSS (le Décor)

Qu'est-ce que c'est ?

C'est le Style. Il contrôle les couleurs, la typographie, et l'espacement. Il nous dit :

  • "Mets ce titre en rouge"
  • "Agrandis un peu ce bouton."

Exemple:

/* Colorer le titre en bleu */
h1 {
  color: var(--color-secondary); 
}
/* Centrer le texte */
p {
  text-align: center;
}
/* Styliser le bouton */
button {
  background-color: var(--color-primary);
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
}

3. JavaScript (Interaction)

Le Code HTML:

<h1>Bienvenue !</h1>
<p id="myText">
  Cliquez le bouton pour changer ce texte.
</p>
<button id="myButton">Cliquez ici</button>

Le Code JavaScript:

// 1. On sélectionne le bouton et le texte
let btn = document.getElementById('myButton');
let txt = document.getElementById('myText');

// 2. On écoute l'événement 'click'
btn.onclick = function() {
  // 3. On change le texte et la couleur
  txt.textContent = "Merci ! Vous avez changé le texte.";
  txt.style.color = "var(--color-primary)";
}
